# All binary rooted trees from Component

This folder contains Nexus files containing all binary rooted trees
from n=2 to n=7 as generated by Rod Page's [Component 2.0] running on
Wine on Linux.

 [Component 2.0]: https://web.archive.org/web/20180512112554/http://taxonomy.zoology.gla.ac.uk/rod/cpw/index.html
